PyFileSync

What is PyFileSync?

PyFileSync is (hypothetically speaking) a set of Python-based tools for file synchronization and data collection over the web. Currently it consists of one component:

  • PyFileServer is a python WSGI based WebDAV server. It is currently in alpha (under development)

PyFileSync (PyFileServer) is developed as part of Google's Summer of Code program mentored by the Python Software Foundation . The project blog for the period is located here.
